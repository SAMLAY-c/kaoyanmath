## 📈 本周学习趋势

### 每日学习统计
```dataview
TABLE WITHOUT ID
  date as "📅 日期",
  length(filter(rows.file.link, !contains(tags, "review"))) as "🆕 新学",
  length(filter(rows.file.link, contains(tags, "review"))) as "🔄 复习",
  length(rows.file.link) as "📊 总计"
FROM "02_原子笔记"
WHERE 
  date >= date(this.file.name) - dur(7 days) AND
  date <= date(this.file.name)
GROUP BY date
SORT date DESC
```

### 本周学习内容分布
```dataview
TABLE WITHOUT ID
  difficulty as "🎯 难度",
  length(rows.file.link) as "📝 数量"
FROM "02_原子笔记"
WHERE 
  date >= date(this.file.name) - dur(7 days) AND
  date <= date(this.file.name)
GROUP BY difficulty
SORT length(rows.file.link) DESC
```

## 🔄 复习追踪系统

### 今天应该复习的笔记
```dataview
TABLE WITHOUT ID
  file.link as "📝 笔记",
  topic as "📚 主题",
  date as "📅 学习日期"
FROM "02_原子笔记"
WHERE 
  date < date(this.file.name) AND
  !contains(tags, "review") AND
  (date + dur(1 days) <= date(this.file.name) OR
   date + dur(7 days) <= date(this.file.name) OR
   date + dur(30 days) <= date(this.file.name))
SORT date ASC
LIMIT 15
```

### 复习进度统计
```dataview
TABLE WITHOUT ID
  "🔴 1天后" as "⏰ 复习周期",
  length(filter(rows.file.link, round((date("<% tp.date.now("YYYY-MM-DD") %>") - date) / (1000 * 60 * 60 * 24), 0) >= 1)) as "📝 待复习数"
FROM "02_原子笔记"
WHERE 
  date < date("<% tp.date.now("YYYY-MM-DD") %>") AND
  !contains(tags, "review") AND
  date(date) + dur(1 days) <= date("<% tp.date.now("YYYY-MM-DD") %>")

UNION

TABLE WITHOUT ID
  "🟡 1周后" as "⏰ 复习周期",
  length(filter(rows.file.link, round((date("<% tp.date.now("YYYY-MM-DD") %>") - date) / (1000 * 60 * 60 * 24), 0) >= 7)) as "📝 待复习数"
FROM "02_原子笔记"
WHERE 
  date < date("<% tp.date.now("YYYY-MM-DD") %>") AND
  !contains(tags, "review") AND
  date(date) + dur(7 days) <= date("<% tp.date.now("YYYY-MM-DD") %>")

UNION

TABLE WITHOUT ID
  "🔵 1个月后" as "⏰ 复习周期",
  length(filter(rows.file.link, round((date("<% tp.date.now("YYYY-MM-DD") %>") - date) / (1000 * 60 * 60 * 24), 0) >= 30)) as "📝 待复习数"
FROM "02_原子笔记"
WHERE 
  date < date("<% tp.date.now("YYYY-MM-DD") %>") AND
  !contains(tags, "review") AND
  date(date) + dur(30 days) <= date("<% tp.date.now("YYYY-MM-DD") %>")
```
